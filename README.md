# Atlanta Braves 2026 WAR Room

An interactive baseball analytics dashboard featuring the Atlanta Braves roster with advanced statistics and analysis.

## Features

- Real-time player statistics
- Interactive charts and visualizations using Recharts
- Hitter and pitcher analysis
- Team standings and WAR progress tracking
- Responsive design with mobile support

## Tech Stack

- React 18
- Vite
- Recharts for data visualization
- CSS3 with custom styling

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build

```bash
npm run build
```
